# Задание 2 — API-пайплайн: данные → LLM → результат (3 балла)

**Цель**: написать рабочий скрипт, который автоматически отправляет данные в LLM через API и получает структурированный ответ.

## Что нужно сделать

Написать скрипт на любом языке (Python, PHP, JavaScript и др.), который выполняет пайплайн:

1. Читает входные данные из CSV-файла или внешнего API.
2. Формирует запрос и отправляет его в LLM через API (OpenAI, Anthropic, Qwen, Xiaomi MiMo и др.).
3. Получает от LLM структурированный ответ в формате JSON.
4. Сохраняет результат в файл JSON.

## Примеры задач (выберите одну или придумайте свою)

- Классификация отзывов: скрипт читает отзывы → LLM определяет тональность (positive/negative/neutral) и тему → результат в CSV.
- Извлечение данных: скрипт читает текстовые описания товаров → LLM извлекает характеристики (цена, бренд, категория) → результат в JSON.
- Саммаризация: скрипт читает новости → LLM генерирует краткое содержание каждой → результат в TXT.

## Формат сдачи

Ссылка на GitHub/Gitlab-репозиторий, содержащий:

- Код скрипта.
- README с описанием задачи, инструкцией запуска, примером входных и выходных данных.
- Файл с реальными результатами работы скрипта (не пустой, не выдуманный) и файл с входными данными, на которых получен результат работы (если это был файл).
- Альтернатива: скринкаст (1–2 мин), демонстрирующий запуск и результат.

## Критерии оценки

| Оценка | Критерии                                                                                                                             |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| 3      | Пайплайн запускается, JSON-ответ от LLM парсится корректно, результат сохранён в файл, README понятен и содержит инструкцию запуска. |
| 2      | Скрипт есть, частично работает или README неполный.                                                                                  |
| 1      | Код есть, но не запускается, или нет выходного файла.                                                                                |
| 0      | Работа не сдана.                                                                                                                     |
